# Marketing-Campaign-Performance-Dashboard

## Overview

This project is a marketing campaign performance dashboard created using Power BI, aimed at analyzing campaign effectiveness and providing insights into revenue, advertising spend, profitability, and return on investment (ROI) across multiple marketing channels.

## Features

### Overview Page:

Provides a comprehensive overview of key marketing performance metrics, including Total Revenue, Total Ad Spend, Profit, and ROI, with interactive visualizations to monitor campaign performance across channels and time.

### Campaign Analysis Page:

Focuses on analyzing individual campaign performance by identifying top-performing campaigns, comparing revenue against advertising spend, evaluating ROI, and measuring conversion rates across different marketing channels to support data-driven marketing decisions.
